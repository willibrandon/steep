DESIGN NOTES
============

1. Graph Panel Heights:
   - Wide terminal (>100): 8 lines per graph (clearer detail)
   - Narrow terminal (<100): 4 lines per graph (compact)
   - Minimum terminal width: 80

2. Time Window Selection:
   - Key 'w' cycles forward through windows (1m, 5m, 15m, 1h, 24h)
   - Key 'W' (shift+w) cycles backward
   - Current window shown in graph caption: "TPS (1h)"
   - Footer shows current window with [w]Window: 1h

3. Graph Colors:
   - TPS: Green (asciigraph.Green)
   - Connections: Blue (asciigraph.Blue)
   - Cache Hit Ratio: Cyan (asciigraph.Cyan)
   - Warning state: Yellow
   - Critical state: Red

4. Data Requirements:
   - Minimum 3 points to render a graph
   - Show "Collecting data..." for insufficient points
   - Show point count progress: "(23/60 points)"

5. Toggle Behavior:
   - 'v' key toggles chart visibility globally
   - State persists across view switches
   - Footer hint changes: "[v]Show Charts" / "[v]Hide Charts"

6. Responsive Layout:
   - Graphs fill available width (width - 4 for borders)
   - Height allocated: 40% of remaining space (after metrics, footer)
   - Minimum graph height: 4 lines
